## ğŸš€ Quick Start

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Swayam26262/Portfolio.git

# Navigate to project directory
cd Portfolio

# Install dependencies
npm install
# or using yarn
yarn install
# or using pnpm
pnpm install
```

### Development

```bash
# Start the development server
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the magic happen! âœ¨
